# Promon

Promon is a Rust-first Node.js process manager for JavaScript and TypeScript projects.

This repository currently contains the first MVP implementation:

- `promon init`
- `promon validate`
- `promon doctor`
- `promon start`
- `promon stop`
- `promon list`

The daemon, service integration, cluster mode, watch mode, log rotation, and TUI are planned in later phases.
